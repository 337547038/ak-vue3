# 使用

## 全局引入（推荐）

将资源包复制到项目，如 src 目录下，然后在入口页面 main.js 中如下配置：

```javascript
import Component from "../packages/index";
// 全局注册基础组件
Vue.use(Component);
```

## 按需引用

在当前页面 import 所需组件

```javascript
import Button from "../packages/button.vue";
```
