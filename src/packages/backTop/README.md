# BackTop 返回顶部

### 用法

```vue demo
<template>
   <div style="height:1500px">滚动页面试试</div>
   <ak-backTop></ak-backTop>
</template>
<script>

</script>
```

## API

### BackTop Props

| 参数      | 类型          | 说明   |
|----------|--------------|--------|
|height    | Number/200     |页面滚动高度达到该值时才显示BackTop组件|
|bottom    | Number/30      |组件距离底部的距离|
|right     | Number/30      |组件距离右部的距离|
|text      | String         |显示文本|
